Thank you for downloading Audovia.  We hope you have found it useful.  A newer version, Audovia 4.0 (Audovia Classic), is now available from the Snap Store.  It can also be installed by using: sudo snap install audovia-classic.

Audovia Classic differs from Audovia in that the "home" interface is now used which means that files are read from and written to directories under: ~/Documents/Audovia/.  You can migrate your files and database from Audovia to Audovia Classic by copying all the directories in: ~/snap/audovia/current/ to: ~/Documents/Audovia/.
