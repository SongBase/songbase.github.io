Make Music with Audovia
 
Audovia 3.1.1 for Microsoft Windows 
Audovia uses JFugue MusicString notation to let you make music with up to 15 instrumental voices and a percussion track.

You can export your songs to MIDI, MusicXML or WAV.

The WAV files can be opened in Audacity and exported to MP3.

Audovia requires Java to be installed.

Features:
  • Database structure for quick and easy editing.

  • Note entry manual or by picking from graphic Treble, Alto, Tenor or Bass staves.

  • Song backup and restore via XML.

  • JFugue extension for string constants.

  • Shared database support for MySQL.
